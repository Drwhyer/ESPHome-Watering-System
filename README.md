# ESPHome-Watering-System
ESPHome-Watering-System is a multi zone garden watering system based on the esp32 and inexpensive 24v Orbit valves.



Parts:
- esp-32
- 8 Channel relay board
- 24v Orbit sprinkler valves (1x per zone)
  https://www.lowes.com/pd/Orbit-0-75-in-Plastic-Electric-Inline-Irrigation-Valve/1000152381
- 24v ac power supply (1x needed)
- Control box
- Weather proof cable grip (2x)
- 3/4 Pex pipe
- 3/4 Pex brass T
- 3/4 Pex brass 3/4 npt
- Water filter
- 3/4 npt male to 3/4 water hose female
